Open CLO Roadmap

윈도우 랩탑에서 리눅스 AI 에이전트 구축까지의 전 과정

Guide v1.0 Production

Step 01. 가상화 기반 다지기

가장 먼저 하드웨어가 리눅스를 받아들일 수 있도록 설정합니다.

1

관리자 권한으로 CMD 실행

2

명령어 입력: wsl --install

3

완료 후 반드시 시스템 재부팅

Step 02. 리눅스 내부 환경 구축

Ubuntu 터미널 내부에 개발 일꾼(Node, Python)을 배치합니다.

Node.js & Python

nvm을 통한 Node 설치 및 apt를 통한 Python 패키지 관리자 설치

VS Code 연동

WSL 확장팩 설치 후 터미널에서 'code .' 입력하여 연결

Step 03. 설치 및 모바일 연동

오픈 클로 에이전트 설치와 텔레그램 페어링 단계입니다.

LLM 연동

Claude / Gemini API Key를 발급받아 .env 파일에 저장

Telegram Pairing

@BotFather 토큰 발급 후 모바일 앱과 연결

Step 04. 웹 접속 및 원격 제어

어디서든 접속 가능한 네트워크 환경을 설정합니다.

네트워크 모드 HOST=0.0.0.0

"집에서는 내부 IP로, 밖에서는 Cloudflare 터널링 주소를 사용하여 내 노트북 속의 AI와 끊김 없이 대화하세요."

Step 05. 페르소나 및 실무 활용

에이전트에게 자아를 부여하고 업무에 투입합니다.

시스템 메모리 설정

에이전트의 이름, 말투, 업무 원칙을 system_prompt.txt에 영구적으로 기록하여 매 대화마다 일관된 성능을 유지하도록 관리합니다.
